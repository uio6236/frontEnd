시맨틱 태그는 의미를 가지고 있는 태그로, 화면에 직접적인 디자인 효과는 없지만 컴퓨터가 웹 페이지 영역을 쉽게 구분하기 위한 용도로 사용됨.

<__> ____ 웹 페이지의 핵심 본문 콘텐츠가 들어가는 시맨틱 영역 ___

서식 태그들

  • b, strong : <_>굵게 표시
  • i, em : <_>기울임 표시
  • u : <_>밑줄 표시
  • s : <_>취소선 표시

영역을 구성하는 요소 종류

<__>
  • block : 블록 요소. 가로줄 전체를 차지하며, 자동으로 줄바꿈이 일어남 <__>
  • div
  • p
  • h1 ~ h6, ...
  • inline : 인라인 요소. 안에 들어있는 콘텐츠(내용) 크기만큼만 공간을 차지하며 옆으로 나열됨 <__>
  • span
  • a, img
  • b, i, ...

    • 표 형식의 태그

    태그 종류 table 하나의 표(테이블)를 표현함
    tr 하나의 행(가로줄)을 표현함
    td, th 하나의 칸(셀)을 표현함
