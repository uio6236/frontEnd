p 태그: 줄바꿈과 공백 무시

HTML은 기본적으로 코드 상의 줄바꿈과 여러 개의 공백을 무시한다. 줄을 바꾸고 싶다면 br 태그를 이용해야 한다.

공백을 여러 칸으로 표현하고자 한다면 특수기호(엔터티코드)를 사용해야 한다.
    공백을 의미하는 코드는 &nbsp; 이다.
수식 표현 : n2 < n1; 처럼 괄호 기호는 특수 기호로 표현해야 안전하다
자주 사용하는 코드: < (미만), > (초과), & (앰퍼샌드)

pre 태그: 작성한 대로 표현

이 곳은 pre 태그 내부이다.
        작성한 내용 그대로, 줄바꿈과    공백이 화면에 출력된다.
        들여쓰기도 그대로 표현된다.

b, strong : 글자를 굵게 표시하는 태그

b: 단순히 글자를 굵게 표시
strong: 글자를 굵게 표시하는데 중요한 내용을 강조 (스크린 리더기가 강조하여 읽음)

i, em : 글자를 기울여서 표시하는 태그

i : 단순히 글자를 기울여서 표시
em : 글자를 기울여서 표시하는데 특정 단어를 강조하기 위해 사용

그 외의 서식 태그

u : 글자 아래에 밑줄을 표시하는 태그
s : 글자 가운데에 줄을 표시하는 태그(취소선)
mark : 글자 배경을 노란색으로 표시하는 태그(형광펜)

sub : 아래첨자로 표시하는 태그 ( H2O )

sup : 윗첨자로 표시하는 태그 (X2)

ul: 순서가 없는 목록

  • Java
  • Oracle
  • HTML

ol: 순서가 있는 목록

  1. Java
  2. Oracle
  3. HTML

ol 태그의 속성

  • type: 표시할 문자들 지정(숫자:1, 알파벳: A, a, 아라비아숫자: I,i)
  • start: 시작 위치(순번)
  • reversed: 역순으로 표시
